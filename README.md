# Matugen Vesktop theme

#### How to install

this assumes you already have matugen and everything setup         
                      
        
put this in `~/.config/matugen/config.toml`               
            
`[templates.vesktop]
input_path = '~/.config/matugen/templates/vesktop/theme.css'           
output_path = '~/.config/vesktop/themes/theme.css'               
`
